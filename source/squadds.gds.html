squadds.gds.processing.bias_gds_features(input_gds, output_gds, bias, layer_number, datatype_number=None)[source]#

Biases features on a specific layer in a GDS file by expanding or contracting them by the specified amount using KLayout.

Parameters:

  • input_gds (str) ‚Äì The path to the input GDS file.

  • output_gds (str) ‚Äì The path to the output GDS file.

  • bias (float) ‚Äì The amount by which to bias the features (in microns). Positive values expand features, negative values contract them.

  • layer_number (int) ‚Äì The layer number of the features to bias.

  • datatype_number (int, optional) ‚Äì The datatype number of the features to bias. If None, all datatypes on the layer are biased.
