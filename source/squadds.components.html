The base TransmonClaw class.

Inherits BaseQubit class.

‚Äòclaw_width‚Äô and ‚Äòclaw_gap‚Äô define the width/gap of the CPW line that makes up the connector. Note, DC SQUID currently represented by single inductance sheet

Add connectors to it using the connection_pads dictionary. See BaseQubit for more information.

BaseQubit Default Options:

  • connection_pads: Empty Dict ‚Äì The dictionary which contains all active connection lines for the qubit.

  • _default_connection_pads: empty Dict ‚Äì The default values for the (if any) connection lines of the qubit.

Default Options:

  • cross_width: ‚Äò20um‚Äô ‚Äì Width of the CPW center trace making up the Crossmon

  • cross_length: ‚Äò200um‚Äô ‚Äì Length of one Crossmon arm (from center)

  • cross_gap: ‚Äò20um‚Äô ‚Äì Width of the CPW gap making up the Crossmon

  • _default_connection_pads: Dict

    • connector_type: ‚Äò0‚Äô ‚Äì 0 = Claw type, 1 = gap type

    • claw_length: ‚Äò30um‚Äô ‚Äì Length of the claw ‚Äòarms‚Äô, measured from the connector center trace

    • ground_spacing: ‚Äò5um‚Äô ‚Äì Amount of ground plane between the connector and Crossmon arm (minimum should be based on fabrication capabilities)

    • claw_width: ‚Äò10um‚Äô ‚Äì The width of the CPW center trace making up the claw/gap connector

    • claw_gap: ‚Äò6um‚Äô ‚Äì The gap of the CPW center trace making up the claw/gap connector

    • connector_location: ‚Äò0‚Äô ‚Äì 0 => ‚Äòwest‚Äô arm, 90 => ‚Äònorth‚Äô arm, 180 => ‚Äòeast‚Äô arm
